**Iteration log**:
- Iteration 1: After first `sign-in.sh` + `pnpm dev:desktop`, log spammed `Error while decrypting the ciphertext provided to safeStorage.decryptString`. Root cause: macOS `safeStorage` derives the keychain entry name from `app.getName()` — by default `"Electron"`, so the helper wrote to `"Electron Safe Storage"` while the desktop reads from `"Lightfast Dev Safe Storage"` (set by `app.setName` in `bootstrap.ts`).
