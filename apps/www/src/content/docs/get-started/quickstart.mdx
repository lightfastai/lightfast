Lightfast is the memory layer for software teams and AI agents. This guide walks through the current setup path: email-based account access, organization creation, and API access.

## Step 1 — Sign in with email

Go to <a href="https://lightfast.ai/sign-in">lightfast.ai</a> and enter your email address. Lightfast sends a one-time verification code to that address. Enter the code to complete sign-in.

New users can create an account from the sign-up page. Invited users should use the invitation link from email, which accepts the team invitation directly after verification.

## Step 2 — Create an organization

Create an organization for your team. An organization is the shared workspace where team membership, API keys, and product access are managed.

Choose a short, stable slug for the organization, such as `acme-platform` or `frontend`. You can rename the organization later from team settings.

## Step 3 — Invite teammates

Add the people who should have access to the organization. Invitations are accepted through email, so teammates do not need a social account or an external identity provider to join.

## Step 4 — Create an API key

For programmatic access, create an organization API key from team settings or use the CLI setup flow. API keys are scoped to the organization that created them.

Use the [TypeScript SDK](/docs/integrate/sdk), [REST API](/docs/api-reference/getting-started/overview), or [MCP server](/docs/integrate/mcp) to connect Lightfast to your own workflows and agents.

<Alert>
Keep API keys server-side. Do not expose organization API keys in browser code, mobile apps, or public repositories.
</Alert>
