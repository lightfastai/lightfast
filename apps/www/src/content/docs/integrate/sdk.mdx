# TypeScript SDK

The `lightfast` package calls the public `/api/v1` resource API with organization-scoped API keys.

## Installation

```bash
npm install lightfast
```

## Quick Start

```typescript
import { createLightfast } from "lightfast";

const lightfast = createLightfast(process.env.LIGHTFAST_API_KEY);

const signal = await lightfast.signals.create({
  input: "Summarize the latest auth architecture decision",
});

console.log(signal.id);
```

## Configuration

```typescript
const lightfast = createLightfast("lf_...", {
  baseUrl: "https://lightfast.ai",
});
```

For tests or local proxies, pass a custom `fetch` implementation:

```typescript
const lightfast = createLightfast("lf_...", {
  fetch: async (request) => fetch(request),
});
```

## Signals

Create a signal:

```typescript
await lightfast.signals.create({
  input: "Check whether the billing connector needs follow-up",
});
```

List signals:

```typescript
const page = await lightfast.signals.list({
  limit: 20,
  statuses: ["queued", "processing", "classified"],
});

for (const item of page.items) {
  console.log(item.id, item.status);
}
```

Fetch a signal:

```typescript
const signal = await lightfast.signals.get({
  id: page.items[0].id,
});

console.log(signal.classification);
```

## Health

```typescript
const health = await lightfast.system.health();
console.log(health.status);
```

## Error Handling

```typescript
import { LightfastApiError, createLightfast } from "lightfast";

const lightfast = createLightfast(process.env.LIGHTFAST_API_KEY);

try {
  await lightfast.signals.create({ input: "" });
} catch (error) {
  if (error instanceof LightfastApiError) {
    console.log(error.status);
    console.log(error.body);
  }
}
```
