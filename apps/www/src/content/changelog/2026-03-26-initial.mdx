Here's everything in v0.1.0.

## Neural Pipeline

The core of Lightfast. Every webhook event flows through four stages:

1. **Ingest** — validates webhook signatures, transforms provider-specific payloads into a canonical format, and stores a delivery log.
2. **Extract** — scores each event for significance, then decomposes it into up to 50 entities: PRs, deployments, issues, endpoints, branches, configs, and more. Each entity is deduplicated and tracked across occurrences.
3. **Graph** — resolves relationships between co-occurring entities and inserts edges with confidence scores. Entities that appear together in the same events become connected.
4. **Embed** — builds a narrative summary for each entity (identity, recent activity, relationships), generates a vector embedding, and upserts it into the search index.

No configuration required. Connect a source, and the pipeline handles everything from there.

## Explore

Ask questions about your engineering infrastructure and get answers grounded in your actual data.

Explore is a streaming interface that searches your entity graph, retrieves relevant context from the vector store, and responds with cited, markdown-formatted answers. You can see exactly which entities and events informed each response through collapsible reasoning traces and tool call cards.

## Source Connections

Connect GitHub, Vercel, Linear, and Sentry in seconds. Each integration uses OAuth with secure token storage — GitHub via App installation tokens, others via standard OAuth 2.0 flows.

Once connected, events flow in real-time through webhooks. Each source shows its status, active resources, and event subscriptions. You choose which repos, projects, and resources to track.

## Entities

A real-time browser for every entity extracted from your engineering events — repositories, pull requests, deployments, issues, users, endpoints, and more.

Filter by category, search by name, and watch new entities appear live as events are processed. Each entity links to a detail view showing its full metadata and event timeline.

Powered by server-sent events — no polling, no refresh.

## Events

A real-time feed of every event from your connected sources. Filter by provider, search by content, and scope by time range. New events stream in as they arrive.

## Public API

A contract-first REST API with interactive documentation. Search entities with semantic reranking, or integrate Lightfast into your own tools and workflows.

Dual authentication: API keys for programmatic access, session tokens for browser-based flows.

## SDK & MCP Server

The TypeScript SDK (`lightfast` on npm) gives you typed access to the full API from any Node.js environment. The MCP server (`@lightfastai/mcp`) exposes the same capabilities to AI agents and tools that support the Model Context Protocol.

```bash
npm install lightfast
```

## API Keys

Create, rotate, and revoke org-scoped API keys from the dashboard. Keys are hashed at rest and shown only once on creation. Rotate atomically — the old key is revoked and a new one is issued in a single operation.
