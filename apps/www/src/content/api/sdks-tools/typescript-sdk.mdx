# TypeScript SDK Reference

The `lightfast` package provides a type-safe client for the public `/api/v1` resource API.

## Installation

```bash
npm install lightfast
```

## Client

```typescript
import { createLightfast } from "lightfast";

const lightfast = createLightfast(process.env.LIGHTFAST_API_KEY);
```

### createLightfast(apiKey, options)

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `apiKey` | `string` | Yes | Organization API key. Must start with `lf_` and include characters after the prefix. |
| `options.baseUrl` | `string` | No | API origin. Defaults to `https://lightfast.ai`. |
| `options.fetch` | `typeof fetch` | No | Custom fetch implementation for tests or proxies. |

## Signals

### signals.create()

Queue a new signal for processing.

```typescript
const signal = await lightfast.signals.create({
  input: "Review the latest authentication architecture changes",
});
```

Returns:

```typescript
{
  id: "signal_...",
  status: "queued",
  visibilityScope: "user"
}
```

### signals.list()

List signals for the API key organization.

```typescript
const page = await lightfast.signals.list({
  limit: 25,
  statuses: ["queued", "processing"],
});
```

`signals.list()` accepts an optional `cursor`, `limit`, and `statuses` array. The response includes `items` and `nextCursor`; pass `nextCursor` back as `cursor` to request the next page.

### signals.get()

Fetch a single signal by id.

```typescript
const signal = await lightfast.signals.get({
  id: "signal_...",
});
```

The response includes the signal input, status, classification, entity links, visibility scope, and timestamps.

## System

### system.health()

Check the authenticated public API health endpoint.

```typescript
const health = await lightfast.system.health();
```

## Errors

The SDK throws `LightfastApiError` for non-2xx responses.

```typescript
import { LightfastApiError, createLightfast } from "lightfast";

try {
  await createLightfast("lf_...").signals.create({ input: "" });
} catch (error) {
  if (error instanceof LightfastApiError) {
    console.log(error.status);
    console.log(error.body);
  }
}
```
