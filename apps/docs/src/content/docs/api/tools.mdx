# Tools API

Manage and execute tools that extend agent capabilities.

## List Available Tools

```http
GET /v1/tools
```

### Response

```json
{
  "data": [
    {
      "id": "web-search",
      "name": "Web Search",
      "description": "Search the web for information",
      "category": "research",
      "parameters": {
        "query": {
          "type": "string",
          "required": true
        }
      }
    }
  ]
}
```

## Register Custom Tool

```http
POST /v1/tools
```

### Request Body

```json
{
  "name": "custom-api",
  "description": "Call custom API endpoint",
  "endpoint": "https://api.example.com/process",
  "method": "POST",
  "headers": {
    "X-API-Key": "${secrets.api_key}"
  },
  "parameters": {
    "data": {
      "type": "object",
      "required": true
    }
  }
}
```

## Execute Tool

```http
POST /v1/tools/{tool_id}/execute
```

### Request Body

```json
{
  "parameters": {
    "query": "latest AI news"
  }
}
```

### Response

```json
{
  "data": {
    "result": {
      "articles": [...]
    },
    "execution_time": 1.23
  }
}
```

## Update Tool

```http
PATCH /v1/tools/{tool_id}
```

## Delete Tool

```http
DELETE /v1/tools/{tool_id}
```
