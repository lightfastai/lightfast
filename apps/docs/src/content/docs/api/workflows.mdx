# Workflows API

Create and manage complex multi-step workflows.

## Create Workflow

```http
POST /v1/workflows
```

### Request Body

```json
{
  "name": "document-processor",
  "description": "Process and analyze documents",
  "steps": [
    {
      "id": "extract",
      "type": "agent",
      "agent_id": "agent_ocr",
      "input": "${input.document}"
    },
    {
      "id": "analyze",
      "type": "agent",
      "agent_id": "agent_nlp",
      "input": "${steps.extract.output}",
      "depends_on": ["extract"]
    }
  ]
}
```

## Execute Workflow

```http
POST /v1/workflows/{workflow_id}/execute
```

### Request Body

```json
{
  "input": {
    "document": "base64_encoded_document"
  },
  "async": true
}
```

### Response

```json
{
  "data": {
    "execution_id": "wf_exec_123",
    "status": "running",
    "steps": {
      "extract": "completed",
      "analyze": "running"
    }
  }
}
```

## Get Workflow Status

```http
GET /v1/workflows/executions/{execution_id}
```

## List Workflows

```http
GET /v1/workflows
```

## Workflow Templates

```http
GET /v1/workflows/templates
```

### Response

```json
{
  "data": [
    {
      "id": "template_rag",
      "name": "RAG Pipeline",
      "description": "Retrieval Augmented Generation workflow",
      "parameters": ["document_source", "chunk_size"]
    }
  ]
}
```
