# API Overview

The Lightfast API provides programmatic access to all platform features. This reference covers REST endpoints, WebSocket connections, and SDK methods.

## Base URL

```
https://api.lightfast.dev/v1
```

## Authentication

All API requests require authentication using an API key:

```bash
curl https://api.lightfast.dev/v1/agents \
  -H "Authorization: Bearer lf_sk_..."
```

## Response Format

```json
{
  "data": {},
  "meta": {
    "request_id": "req_abc123",
    "timestamp": "2024-01-10T10:00:00Z"
  }
}
```

## Rate Limits

| Tier | Requests/min | Tokens/day |
|------|-------------|------------|
| Free | 60 | 100,000 |
| Pro | 600 | 1,000,000 |
| Enterprise | Custom | Custom |

## Common Endpoints

### Agents
- `GET /agents` - List agents
- `POST /agents` - Create agent
- `GET /agents/{id}` - Get agent
- `DELETE /agents/{id}` - Delete agent

### Executions
- `POST /agents/{id}/execute` - Run agent
- `GET /executions/{id}` - Get execution status
- `POST /executions/{id}/cancel` - Cancel execution

### Tools
- `GET /tools` - List available tools
- `POST /tools` - Register custom tool

## Error Handling

```json
{
  "error": {
    "code": "rate_limit_exceeded",
    "message": "Rate limit exceeded",
    "details": {
      "limit": 60,
      "remaining": 0,
      "reset_at": "2024-01-10T10:05:00Z"

  }

```

## SDK Examples

### JavaScript
```javascript
const { Lightfast } = require('@lightfast/sdk');
const client = new Lightfast({ apiKey: 'lf_sk_...' });
```

### Python
```python
from lightfast import Lightfast
client = Lightfast(api_key='lf_sk_...')
```

### Go
```go
import "github.com/lightfast/lightfast-go"
client := lightfast.NewClient("lf_sk_...")
```
