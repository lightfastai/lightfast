# Quick Start Guide

Get your first agent running in minutes with this step-by-step guide.

## Prerequisites

- Node.js 18+ or Python 3.9+
- npm/yarn/pnpm or pip
- A Lightfast account (sign up at [lightfast.dev](https://lightfast.dev))

## Installation

### JavaScript/TypeScript

```bash
npm install @lightfast/sdk
# or
yarn add @lightfast/sdk
# or
pnpm add @lightfast/sdk
```

### Python

```bash
pip install lightfast
```

## Authentication

Get your API key from the [dashboard](https://app.lightfast.dev/api-keys).

### Set Environment Variable

```bash
export LIGHTFAST_API_KEY="lf_sk_..."
```

### Or Configure in Code

```typescript
import { Lightfast } from '@lightfast/sdk';

const client = new Lightfast({
  apiKey: 'lf_sk_...'
});
```

## Your First Agent

### Simple Example

```typescript
import { Agent } from '@lightfast/sdk';

const agent = new Agent({
  name: 'assistant',
  model: 'gpt-4',
  instructions: 'You are a helpful assistant.'
});

const response = await agent.run({
  message: 'What is the weather in San Francisco?'
});

console.log(response.content);
```

### With Tools

```typescript
const agent = new Agent({
  name: 'researcher',
  model: 'gpt-4',
  tools: ['web-search', 'calculator'],
  instructions: 'Research topics and provide detailed analysis.'
});

const result = await agent.run({
  message: 'Find the latest AI research papers on reasoning'
});
```

## Common Patterns

### Streaming Responses

```typescript
const stream = await agent.stream({
  message: 'Write a story about robots'
});

for await (const chunk of stream) {
  process.stdout.write(chunk.content);
}
```

### Error Handling

```typescript
try {
  const response = await agent.run({ message });
} catch (error) {
  if (error.code === 'RATE_LIMIT') {
    // Handle rate limiting
  } else if (error.code === 'TIMEOUT') {
    // Handle timeout
  }
}
```

### Conversation Management

```typescript
const conversation = agent.createConversation();

await conversation.send('Hello');
await conversation.send('What did I just say?');
// Agent remembers context
```
