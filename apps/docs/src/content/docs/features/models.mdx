# AI Models

Lightfast Chat supports multiple AI models to suit different needs and use cases.

## Available Models

### Claude Models
- **Claude 3.5 Sonnet**: Latest model with enhanced capabilities
- **Claude 3 Opus**: Most capable model for complex tasks
- **Claude 3 Haiku**: Fast and efficient for simple tasks

### OpenAI Models  
- **GPT-4**: Advanced reasoning and generation
- **GPT-4 Turbo**: Faster variant with updated knowledge
- **GPT-3.5 Turbo**: Cost-effective for general use

## Model Selection

Switch between models seamlessly within your conversation using the model selector dropdown.

## Configuration

Add your API keys in Settings to enable different model providers:
- Anthropic API key for Claude models
- OpenAI API key for GPT models
