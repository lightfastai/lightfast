# Chat Features

Explore the powerful chat capabilities of Lightfast Chat.

## Message Streaming

Experience real-time AI responses with smooth character-by-character streaming, providing immediate feedback as the AI generates its response.

## Multi-turn Conversations

Engage in extended dialogues with context preservation across multiple messages. The AI maintains conversation history for coherent interactions.

## Code Highlighting

Share code snippets with automatic syntax highlighting for over 100 programming languages, making technical discussions clear and readable.

## Markdown Support

Full markdown rendering including:
- Headers and formatting
- Lists and tables
- Links and images
- Code blocks with syntax highlighting

## File Attachments

Upload and share images directly in your conversations. Supported formats include:
- JPEG/JPG
- PNG
- GIF
- WebP

## Message Actions

- Copy messages to clipboard
- Edit your messages
- Delete messages
- Share individual messages

## Keyboard Shortcuts

- `Enter` - Send message
- `Shift + Enter` - New line
- `Cmd/Ctrl + K` - New chat
- `Cmd/Ctrl + /` - Toggle sidebar
