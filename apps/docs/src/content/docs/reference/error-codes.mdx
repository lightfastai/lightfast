# Error Code Reference

Comprehensive list of error codes and their solutions.

## Authentication Errors (1xxx)

| Code | Error | Description | Solution |
|------|-------|-------------|----------|
| 1001 | `AUTH_INVALID_KEY` | Invalid API key | Check your API key |
| 1002 | `AUTH_EXPIRED_KEY` | Expired API key | Renew your API key |
| 1003 | `AUTH_INSUFFICIENT_PERMISSIONS` | Insufficient permissions | Upgrade plan or request access |
| 1004 | `AUTH_RATE_LIMITED` | Rate limit exceeded | Implement backoff |

## Execution Errors (2xxx)

| Code | Error | Description | Solution |
|------|-------|-------------|----------|
| 2001 | `EXEC_TIMEOUT` | Execution timeout | Increase timeout or optimize |
| 2002 | `EXEC_INVALID_INPUT` | Invalid input format | Validate input schema |
| 2003 | `EXEC_TOOL_ERROR` | Tool execution failed | Check tool configuration |
| 2004 | `EXEC_MODEL_ERROR` | Model error | Retry or use fallback model |

## Resource Errors (3xxx)

| Code | Error | Description | Solution |
|------|-------|-------------|----------|
| 3001 | `RESOURCE_QUOTA_EXCEEDED` | Quota exceeded | Upgrade plan |
| 3002 | `RESOURCE_MEMORY_LIMIT` | Memory limit reached | Optimize memory usage |
| 3003 | `RESOURCE_CPU_LIMIT` | CPU limit reached | Optimize processing |
| 3004 | `RESOURCE_STORAGE_FULL` | Storage full | Clean up or upgrade |

## Network Errors (4xxx)

| Code | Error | Description | Solution |
|------|-------|-------------|----------|
| 4001 | `NETWORK_TIMEOUT` | Network timeout | Check connectivity |
| 4002 | `NETWORK_DNS_FAILURE` | DNS resolution failed | Verify domain |
| 4003 | `NETWORK_CONNECTION_REFUSED` | Connection refused | Check firewall |
| 4004 | `NETWORK_SSL_ERROR` | SSL/TLS error | Verify certificates |

## Validation Errors (5xxx)

| Code | Error | Description | Solution |
|------|-------|-------------|----------|
| 5001 | `VALIDATION_SCHEMA` | Schema validation failed | Fix data format |
| 5002 | `VALIDATION_TYPE` | Type mismatch | Check data types |
| 5003 | `VALIDATION_REQUIRED` | Required field missing | Provide all required fields |
| 5004 | `VALIDATION_CONSTRAINT` | Constraint violation | Meet constraints |

## System Errors (9xxx)

| Code | Error | Description | Solution |
|------|-------|-------------|----------|
| 9001 | `SYSTEM_INTERNAL` | Internal error | Contact support |
| 9002 | `SYSTEM_MAINTENANCE` | Under maintenance | Wait and retry |
| 9003 | `SYSTEM_DEGRADED` | Degraded performance | Monitor status page |
| 9999 | `SYSTEM_UNKNOWN` | Unknown error | Contact support |

```json
  "error": {
    "code": 2001,
    "type": "EXEC_TIMEOUT",
    "message": "Execution exceeded timeout of 30s",
    "details": {
      "executionId": "exec_123",
      "duration": 30001,
      "timeout": 30000
    },
    "timestamp": "2024-01-10T10:00:00Z"
  }
}
```

## Handling Errors

```typescript
try {
  const result = await agent.execute({ input });
} catch (error) {
  switch (error.code) {
    case 1004: // Rate limited
      await sleep(error.retryAfter);
      return retry();
      
    case 2001: // Timeout
      return handleTimeout();
      
    case 3001: // Quota exceeded
      return notifyUserToUpgrade();
      
    default:
      logger.error('Unexpected error', error);
      throw error;
  }
}
```

Know your errors, handle them gracefully!
