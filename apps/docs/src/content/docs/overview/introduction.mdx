## Technology Stack

Built on a foundation of modern, production-ready technologies:

- **Frontend**: Next.js 15 with App Router and React 19 RC
- **Backend**: Convex for real-time database and serverless functions
- **Styling**: Tailwind CSS v4 with shadcn/ui components (New York)
- **Language**: TypeScript 5.8 with strict mode
- **Runtime**: Bun 1.2.10 for blazing-fast performance
- **Build**: Turborepo 2.5 for optimized monorepo builds
- **Quality**: Biome for fast linting and formatting
- **Auth**: Convex Auth with GitHub OAuth integration
