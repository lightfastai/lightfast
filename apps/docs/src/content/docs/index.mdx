# Lightfast Documentation

Lightfast is **the memory layer for software teams**. We help people and agents find what they need, understand context, and trace decisions across your entire organization.

## What is Lightfast for developers?

Lightfast gives developers a simple API and MCP tools to treat their organization&apos;s knowledge as shared memory. Use four routes&mdash;`/v1/search`, `/v1/contents`, `/v1/similar`, and `/v1/answer`&mdash;to add semantic search with sources to apps, agents, and internal tools in minutes.

## What Lightfast Does

- **Search by meaning**: Find relevant information based on intent, not just keywords
- **Always cite sources**: Every answer shows where it came from
- **Explainable by design**: See who owns what, what depends on what, and why decisions were made
- **Developer-first**: Four simple API routes. MCP tools for agents. Integrate in minutes.
