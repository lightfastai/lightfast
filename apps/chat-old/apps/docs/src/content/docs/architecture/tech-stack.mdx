# Tech Stack

A comprehensive overview of the technologies used in Lightfast Chat.

## Frontend Technologies

### Core Framework
- **Next.js 15**: React framework with App Router
- **React 19**: UI library with Server Components
- **TypeScript 5.8**: Type-safe development

### Styling & UI
- **Tailwind CSS v4**: Utility-first CSS framework
- **shadcn/ui**: Beautifully designed components
- **Radix UI**: Accessible component primitives
- **Lucide Icons**: Beautiful open-source icons

### State Management
- **Convex React**: Real-time data synchronization
- **React Hooks**: Built-in state management
- **Optimistic Updates**: Instant UI feedback

## Backend Technologies

### Database & API
- **Convex**: Reactive database and serverless functions
- **TypeScript**: End-to-end type safety
- **WebSockets**: Real-time communication

### AI Integration
- **Anthropic SDK**: Claude model integration
- **OpenAI SDK**: GPT model integration
- **Streaming**: Real-time response streaming

### Authentication
- **NextAuth.js**: Authentication framework
- **GitHub OAuth**: Social authentication
- **JWT**: Secure token handling

## Development Tools

### Build & Deploy
- **Turborepo**: Monorepo build system
- **pnpm**: Fast package manager
- **Vercel**: Frontend hosting platform
- **GitHub Actions**: CI/CD pipelines

### Code Quality
- **Biome**: Fast formatter and linter
- **TypeScript ESLint**: Type-aware linting
- **Prettier**: Code formatting (via Biome)

### Developer Experience
- **Hot Module Replacement**: Instant updates
- **TypeScript**: Compile-time type checking
- **VS Code**: Recommended IDE with extensions

## Infrastructure

### Hosting
- **Vercel**: Frontend hosting with edge functions
- **Convex**: Backend hosting with global distribution
- **CDN**: Static asset delivery

### Monitoring
- **Vercel Analytics**: Performance monitoring
- **Error Tracking**: Built-in error handling
- **Logging**: Structured logging system

## Package Management

### Dependencies
- **pnpm**: Primary package manager
- **npm**: Fallback compatibility
- **Workspaces**: Monorepo package management

### Version Control
- **Git**: Source control
- **GitHub**: Code hosting and collaboration
- **Conventional Commits**: Standardized commit messages
