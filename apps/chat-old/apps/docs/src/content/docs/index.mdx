# Welcome to Lightfast Chat

Welcome to the official documentation for **Lightfast Chat**, an open-source, production-ready AI chat application featuring Claude Sonnet 4, GPT-4o, real-time messaging, and modern UI. Built with Next.js 15, Convex, and TypeScript.
