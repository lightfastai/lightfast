# Quickstart

Get Lightfast Chat running in under 5 minutes with this quickstart guide.

## Prerequisites

Before starting, ensure you have:
- **pnpm 8.0+** installed ([install here](https://pnpm.io/installation))
- A terminal/command prompt
- API keys from at least one AI provider

## 1. Clone and Install

```bash
# Clone the repository
git clone https://github.com/lightfastai/chat.git
cd chat

# Install dependencies with pnpm
pnpm install
```

## 2. Quick Configuration

Create a minimal environment configuration:

```bash
# Navigate to the main app
cd apps/www

# Create .env.local with your API keys
cat > .env.local << 'EOF'
# Convex (local development)
NEXT_PUBLIC_CONVEX_URL=http://127.0.0.1:3210

# AI Providers (add your keys)
ANTHROPIC_API_KEY=sk-ant-your-key-here
OPENAI_API_KEY=sk-your-key-here
OPENROUTER_API_KEY=sk-or-your-key-here
EXA_API_KEY=your-exa-key-here

# Auth (generate a random key)
JWT_PRIVATE_KEY=your-random-secret-key
JWKS={}
EOF
```

## 3. Start the Backend

```bash
# From the root directory
pnpm run convex:dev
```

This will:
- Start Convex development server
- Create database schema
- Deploy serverless functions

## 4. Start the Application

```bash
# In a new terminal, from root directory
pnpm run dev:www

# Or run everything together
pnpm run dev:all
```

## 5. Access the Application

Open your browser and navigate to:
```
http://localhost:3000
```

## Your First Chat

### 1. Sign In (Optional)
For development, you can use the app without signing in. For full features:
- Click "Sign in with GitHub"
- Authorize the application
- You'll be redirected back to the chat

### 2. Start Chatting
- Type your message in the input field at the bottom
- Press Enter or click the send button
- Watch as the AI response streams in real-time

### 3. Try Key Features

**Switch AI Models**
- Click the model selector dropdown (default: Claude Sonnet 4)
- Choose from available models:
  - Claude Sonnet 4 (most capable)
  - GPT-4o-mini (fast and efficient)
  - GPT-4o (OpenAI's best)
- Send another message to compare responses

**Upload an Image**
- Click the paperclip icon
- Select an image file
- Add a message about the image
- Send to get AI analysis

**Share a Conversation**
- Click the share button in the top bar
- Copy the public link
- Open in an incognito window to view

## Quick Customization

### Change the Site Name

Edit `apps/www/src/lib/site-config.ts`:
```typescript
export const siteConfig = {
  name: "Your Chat App",
  // ... other config
}
```

### Add Your Logo

Replace the logo file:
```bash
cp your-logo.svg apps/www/public/logo.svg
```

### Customize Colors

Edit `apps/www/src/app/globals.css`:
```css
:root {
  --primary: 220 90% 56%;  /* Change primary color */
}
```

## Development Tips

### Hot Reload
Changes to your code automatically reload the browser. Try editing:
- `apps/www/src/app/page.tsx` - Home page
- `apps/www/src/components/chat/chat-interface.tsx` - Chat UI

### View Logs
```bash
# Convex function logs
pnpm run convex logs

# Application logs
# Check your terminal running pnpm run dev:www
```

### Reset Database
```bash
# Clear all data (development only)
pnpm run convex run --no-push setup:clearAll
```

## Common Quickstart Issues

### "Convex connection failed"
- Ensure `pnpm run convex:dev` is running
- Check `NEXT_PUBLIC_CONVEX_URL` is set to `http://127.0.0.1:3210`

### "No AI response"
- Verify your AI API key is correct
- Check you have credits/quota with your AI provider

### "Build errors"
```bash
# Quick fix for most build issues
SKIP_ENV_VALIDATION=true pnpm run build
```

## Quick Commands Reference

```bash
# Development (from root)
pnpm run dev:www          # Start Next.js dev server
pnpm run dev:all          # Start Next.js + Convex together
pnpm run build:www    # Build for production
pnpm run lint         # Run linter
pnpm run format       # Format code

# Convex
pnpm run convex:dev   # Start Convex dev
pnpm run convex:deploy # Deploy to production
pnpm run convex logs  # View function logs

# Monorepo
pnpm run dev              # Run all apps
pnpm run build        # Build all apps
pnpm run typecheck    # Type check all packages
```
