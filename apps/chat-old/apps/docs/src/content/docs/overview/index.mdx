# Introduction

Welcome to **Lightfast Chat**, a modern AI-powered chat application built with cutting-edge web technologies.

## What is Lightfast Chat?

Lightfast Chat is a full-featured chat application that provides:

- **Multiple AI Models**: Support for Claude, GPT-4, and other leading AI models
- **Real-time Streaming**: Smooth, responsive chat experiences with streaming responses
- **File Uploads**: Share images and documents in your conversations
- **Conversation Management**: Organize chats into threads with persistent history
- **Sharing**: Share conversations with others via public links
- **Authentication**: Secure sign-in with GitHub OAuth

## Key Features

### AI-Powered Conversations
Engage with state-of-the-art AI models including Claude 3.5 Sonnet, GPT-4, and more. Switch between models seamlessly within the same conversation.

### Real-time Experience
Built on Convex for real-time data synchronization, ensuring instant updates across all connected clients.

### Modern Stack
Leveraging Next.js 15, React 19, and TypeScript for a robust, type-safe development experience.

### Beautiful UI
Crafted with shadcn/ui components and Tailwind CSS for a clean, responsive interface.

## Getting Started

## Architecture Overview

Lightfast Chat is built as a monorepo using Turborepo, with:

- **Frontend**: Next.js app with App Router
- **Backend**: Convex for real-time database and serverless functions
- **Documentation**: This Fumadocs-powered site

Learn more in our [Architecture section](/docs/architecture/overview).
