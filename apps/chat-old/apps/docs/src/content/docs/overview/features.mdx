# Features

Lightfast Chat provides a comprehensive set of features designed for both end users and developers.

## Chat Features

### Multi-Model AI Support
- **Claude Sonnet 4**: Anthropic's most advanced model (default)
- **GPT-4o**: OpenAI's flagship multimodal model
- **GPT-4o-mini**: Fast and cost-effective for most tasks
- **OpenRouter Integration**: Access to 100+ models including Llama, Mixtral, and more
- **Model Switching**: Change models mid-conversation with preserved context

### Real-time Streaming
- **Token Streaming**: See responses as they're generated
- **Smooth Rendering**: Optimized text rendering for readable streaming
- **Progress Indicators**: Visual feedback during generation
- **Cancellation**: Stop generation at any time

### Enhanced Capabilities
- **Web Search**: Integrated Exa search for real-time information
- **Token Usage Tracking**: Monitor API costs per message
- **Model Context**: Full conversation context for better responses
- **Markdown Support**: Rich formatting in AI responses

### Conversation Management
- **Thread Organization**: Each conversation is a separate thread
- **Sidebar Navigation**: Quick access to all your chats
- **Auto-Titling**: AI generates titles for your conversations
- **Persistent History**: All conversations saved automatically
- **Delete Threads**: Remove conversations you no longer need

### File Support
- **Image Uploads**: Share images for AI analysis (PNG, JPG, WebP)
- **Drag & Drop**: Simply drag files into the chat
- **File Preview**: See uploaded images inline
- **Size Limits**: Up to 10MB per file

### Sharing & Collaboration
- **Public Links**: Share conversations with unique URLs
- **Read-Only Access**: Shared chats are view-only for privacy
- **No Sign-In Required**: Anyone with the link can view
- **Automatic Expiry**: Optional expiration for shared links

## User Experience

### Modern Interface
- **Responsive Design**: Optimized for desktop and mobile
- **Dark Mode**: Beautiful dark theme by default
- **Keyboard Shortcuts**: Enter to send, Shift+Enter for new line
- **Clean UI**: Minimalist design focused on conversation

### Performance
- **Instant Loading**: Server-side rendering for fast initial load
- **Partial Prerendering**: Next.js 15 PPR for optimal performance
- **Real-time Updates**: Convex provides instant message updates
- **Optimistic UI**: Messages appear instantly while sending

### Customization
- **Themes**: Choose from multiple color schemes
- **Layout Options**: Customize sidebar, chat width, and more
- **Font Settings**: Adjust text size and font family
- **Notification Preferences**: Control how you're notified

## Developer Features

### Architecture
- **Monorepo Structure**: Organized with Turborepo for scalability
- **Type Safety**: Full TypeScript coverage with strict mode
- **Component Library**: Reusable UI components with shadcn/ui
- **API Design**: RESTful and real-time APIs

### Deployment Options
- **Vercel**: Optimized for Vercel deployment
- **Self-Hosting**: Full instructions for self-deployment
- **Convex Backend**: Serverless backend with global distribution
- **Environment Flexibility**: Works with any Node.js host

### Extensibility
- **Plugin System**: Add custom functionality
- **Webhook Support**: Integrate with external services
- **API Access**: Programmatic access to all features
- **Custom Models**: Add your own AI model providers

### Development Tools
- **Hot Reload**: Instant updates during development
- **Type Generation**: Automatic TypeScript types from database
- **Testing Suite**: Comprehensive unit and integration tests
- **Documentation**: Auto-generated API documentation

## Security & Privacy

### Authentication
- **GitHub OAuth**: Sign in with your GitHub account
- **JWT Sessions**: Secure session management
- **API Key Storage**: Encrypted storage for personal API keys
- **Guest Mode**: Try without signing in (limited features)

### Data Protection
- **Encrypted Storage**: API keys encrypted at rest
- **No Training**: Your conversations aren't used for AI training
- **Data Export**: Download your conversation history
- **Account Deletion**: Remove all data on request

### Access Control
- **Role-Based Access**: Fine-grained permissions
- **Team Management**: Organize users into teams
- **Guest Access**: Temporary access for external users
- **IP Whitelisting**: Restrict access by IP address

## Integrations

### AI Providers
- **Anthropic**: Native Claude integration
- **OpenAI**: Full GPT model support
- **OpenRouter**: Access to 100+ models
- **Custom Providers**: Add your own AI backends

### Third-Party Services
- **Analytics**: Google Analytics, Plausible, etc.
- **Error Tracking**: Sentry integration
- **Search**: Algolia for advanced search
- **Storage**: S3-compatible object storage

### Communication
- **Webhooks**: Send notifications to external services
- **Email**: Transactional email support
- **Slack**: Post updates to Slack channels
- **Discord**: Community integration
