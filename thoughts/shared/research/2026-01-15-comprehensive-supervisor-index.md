# Comprehensive Australian Supervisor Index

**Date**: 2026-01-15
**Purpose**: Complete index of potential MPhil/PhD supervisors across Australian universities, ranked by alignment with Lightfast's agent memory architecture

## Lightfast Architecture Alignment Criteria

For ranking purposes, supervisors are evaluated against Lightfast's core technical contributions:

1. **Multi-view embeddings** (title/content/summary vectors)
2. **Four-path hybrid retrieval** (dense + lexical + cluster + actor)
3. **Hierarchical memory abstraction** (chunks → observations → clusters → profiles)
4. **Implicit knowledge graph** via typed references
5. **Cross-platform identity resolution**
6. **Grounded answer generation** with source citations
7. **Software engineering domain** (code, PRs, incidents, decisions)
